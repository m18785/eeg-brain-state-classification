# eeg-brain-state-classification
EEG signal classification using machine learning

# Классификация состояний мозга по сигналам ЭЭГ

Проект посвящён исследованию возможности классификации состояний мозга по сигналам электроэнцефалограммы (ЭЭГ) с использованием методов машинного обучения.

В работе были проведены три эксперимента с различными моделями и постановками задачи.

---

## Цель проекта

Исследовать возможность распознавания различных состояний мозга по сигналам ЭЭГ с использованием методов машинного обучения.

---

## Используемые технологии

Python  
NumPy  
Pandas  
Matplotlib  
SciPy  
Scikit-learn  
PyTorch  

---

# Эксперименты

## Эксперимент 1 — классификация четырёх состояний

Модель: нейронная сеть (SimpleEEGNet)

Использовались спектральные признаки сигнала ЭЭГ.

Ноутбук:  
eeg1.ipynb

---

## Эксперимент 2 — классификация мысленных образов

Задача: различить

- яблоко
- торт
- шум

Модель: RandomForestClassifier

Ноутбук:  
eeg2.ipynb

---

## Эксперимент 3 — классификация состояния глаз

Задача:

- глаза открыты
- глаза закрыты

Сигнал разбивался на окна, после чего вычислялись мощности диапазонов:

- delta
- theta
- alpha
- beta

Модель: RandomForestClassifier

Полученная точность:

**Accuracy ≈ 0.91**

Ноутбук:  
eeg_final.ipynb

---

# Алгоритм обработки сигнала

1. Запись сигнала ЭЭГ  
2. Разбиение сигнала на временные окна  
3. Преобразование сигнала в частотную область (метод Welch)  
4. Вычисление мощностей диапазонов EEG:
   - delta
   - theta
   - alpha
   - beta
5. Формирование признаков
6. Классификация с использованием модели машинного обучения
